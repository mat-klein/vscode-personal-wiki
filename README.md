# personalWiki README

This extension adds a personal wiki to vscode

## Features

- create a personal wiki
- add/edit wikipages
- use markdown to style your wikipages
- wikilinks to navigate through your content
- add/use templates to define and re-use markdown content
- import/export pages of your personal wiki
